# ZEDTAF

ZED TEST AUTOMATION FRAMEWORK

## Installation

Use the node and npm package manager to install dependencies.

```bash
git clone ....
```

## Usage

```cmd

# Install dependencies
npm install

# Run all the tests
npm run test

# Generated the Allure Report
allure generate
```
